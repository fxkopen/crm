
# 竞争对手

---

商业竞争日趋激烈，销售过程中，每个商机都可能存在多个对手参与竞争，因此关注每个竞争对手的动态及其销售活动，并及时进行积极应对是赢得商机的重要手段。


## 业务使用场景举例
- 商机因竞争对手输单时，可关联竞争对手，方便输单原因追踪及后续分析。
- 可维护竞争对手信息及资料并分析在哪个区域哪个产品或商机上遇到的。


# 竞争对手的详细介绍



## 竞争对手的业务操作
### 竞争对手预设场景
- **全部**：全部竞争对手
- **我负责的**：负责人为当前用户

> 备注：
> - “CRM管理员”可以查看所有数据。
> - 所有状态为“作废”的记录只有“CRM管理员”可见，其他人员均不可见。

### 新建竞争对手
- 有多种方式创建线索。
    - 手工创建入口：在【竞争对手】的主列表下新建竞争对手。
